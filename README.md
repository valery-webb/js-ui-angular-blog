js-ui-angular-blog
==================

1. Develop:
  * Clone this repo
  * Install nodejs
  * Install grunt
  * Make build by command 'grunt'
  * Watch less by command 'grunt run-less'

2. Use:
  * Just clone this repo and run index.html via localhost

<hr />
compatibility: IE8+
<br />
custom grid system: utils.less
<br />
polyfills: html5shiv, respondjs
